# 🎥 Edge AI Proctoring System

A browser-based Edge AI Proctoring application built using React, Vite and Google MediaPipe.

## Features

- 🎥 Live Webcam Monitoring
- 👤 Face Detection
- 👥 Multiple Face Detection
- 👀 Looking Away Detection
- 📊 Live Status Dashboard
- 📋 Event Logs
- 🚨 Alert Counter
- ⚡ Browser-based AI Processing
- 🔒 Privacy Friendly (Edge AI)

## Tech Stack

- React
- Vite
- MediaPipe Tasks Vision
- JavaScript

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
